## The Problem

Traditional grants require: nonprofit status, fiscal sponsors, 30-page applications, 6-month waiting periods, board approval, restricted use, extensive reporting, relationships with funders who already know you.

All of that is gatekeeping dressed as process.

## How SolarPunk Grants Works

**Apply:** Open a GitHub Issue using the application template. 500 words max. What do you need, what for, what will it do. No 30-page applications.

**Vote:** The community votes via GitHub reactions. Thumbs up = support. 7-day window.

**Fund:** Top voted applications each cycle receive funds from the pool. Transparent on-chain tracking. No bureaucracy.

**Report:** Post an update Issue when you do the thing. That's the reporting requirement.
