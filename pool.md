## How Grants Are Awarded

1. Anyone applies via GitHub Issue (500 words max)
2. Community votes via 👍 reactions for 7 days
3. Top-voted applications funded from pool, in order, until pool is exhausted
4. Recipient posts update Issue when project complete
5. Cycle repeats

## What Gets Funded

Anything that makes the world work better for people who need it to.

No restrictions by:
- Cause (Gaza, Sudan, DRC, local, global, environmental, human, all of it)
- Location (anywhere on Earth)
- Person (individuals, collectives, no nonprofit status required)
- Amount (whatever the project needs, up to pool balance)
- Type (art, food, tools, code, legal, medical, housing, anything)
